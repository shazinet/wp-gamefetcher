== Description ==

# WP GameFetcher

WP GameFetcher is a powerful WordPress plugin that seamlessly integrates with the rawg.io API, allowing you to effortlessly fetch and display detailed information about your favorite games directly in your WordPress posts and pages.

## Features

- **Dynamic Game Data:** Display up-to-date details about games, including titles, genres, release dates, and more.
- **Easy Integration:** Utilize the simple shortcode to effortlessly embed game information within your content.
- **Customization:** Tailor the display to suit your website's design with customizable templates and styling options.
- **User-Friendly:** No coding skills required; get started with ease to enhance your gaming content.

## How to Use

### Widget

1. Install and activate the plugin.
2. Go to your WordPress dashboard and navigate to "Appearance" > "Widgets."
3. Drag the "WP GameFetcher Widget" into your desired widget area.
4. Optionally, add a title for the widget.
5. Save your changes, and the game data will be displayed in the widget area.

### Gutenberg Block

1. Install and activate the plugin.
2. Create or edit a post/page using the WordPress block editor.
3. Search for "WP GameFetcher Block" in the block inserter.
4. Add the block to your content.
5. Customize the ordering and page size using the block settings.
6. Save or update your post/page, and the game data will be embedded.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-gamefetcher` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WP GameFetcher to configure the plugin

== Frequently Asked Questions ==

= How do I get started with WP GameFetcher? =

To get started with WP GameFetcher, follow these simple steps:
1. Install and activate the plugin.
2. Configure your rawg.io API key in the plugin settings.
3. Use the provided shortcode in your posts or pages to showcase game data.

= Where can I find my rawg.io API key? =

You can obtain your rawg.io API key by registering on the rawg.io website and creating a new API key in your account settings.

= Can I customize the appearance of the game cards? =

Yes, you can! WP GameFetcher provides customization options. Additionally, you can use your own styling to tailor the display to match your website's design.

= How can I add game data to my widget area? =

Adding game data to your widget area is easy:
1. Install and activate the plugin.
2. Go to your WordPress dashboard and navigate to "Appearance" > "Widgets."
3. Drag the "WP GameFetcher Widget" into your desired widget area.
4. Optionally, add a title for the widget.
5. Save your changes, and the game data will be displayed in the widget area.

= Can I display game data in the WordPress block editor? =

Absolutely! WP GameFetcher comes with a Gutenberg Block. Simply search for "WP GameFetcher Block" in the block inserter, add the block to your content, and customize the ordering and page size using the block settings.

= Is WP GameFetcher translation ready? =

Yes, WP GameFetcher is translation ready. The plugin comes with translations for various languages, including Armenian. You can easily translate or contribute to translations using translation tools like Loco Translate.

= How frequently is the game data updated? =

The game data is fetched from the rawg.io API, and its freshness depends on the data provided by rawg.io. Typically, the data is updated regularly to ensure accuracy.

= Can I use WP GameFetcher on multiple pages? =

Certainly! You can use the WP GameFetcher shortcode or Gutenberg block on as many pages or posts as you like to showcase game data.

= I'm encountering issues. Where can I get support? =

For support, please visit the [WP GameFetcher support forum](https://wordpress.org/support/plugin/wp-gamefetcher/) on WordPress.org. Our community and developers are there to assist you.
