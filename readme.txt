== Description ==

# WP GameFetcher

WP GameFetcher is a powerful WordPress plugin that seamlessly integrates with the rawg.io API, allowing you to effortlessly fetch and display detailed information about your favorite games directly in your WordPress posts and pages.

## Features

- **Dynamic Game Data:** Display up-to-date details about games, including titles, genres, release dates, and more.
- **Easy Integration:** Utilize the simple shortcode to effortlessly embed game information within your content.
- **Customization:** Tailor the display to suit your website's design with customizable templates and styling options.
- **User-Friendly:** No coding skills required; get started with ease to enhance your gaming content.

## How to Use

1. Install and activate the plugin.
2. Configure your rawg.io API key in the settings.
3. Use the provided shortcode to showcase game data wherever you want.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-gamefetcher` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WP GameFetcher to configure the plugin
